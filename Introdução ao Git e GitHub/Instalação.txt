## Instalando no Linux

Se você deseja instalar o Git no Linux através de um instalador binário, você pode geralmente fazêlo
através da ferramenta básica de gerenciamento de pacotes que vem com sua distribuição. Se
você usar Fedora por exemplo, você pode usar o yum:

**$ sudo yum install git-all**

Se você usar uma distribuição baseada em Debian como o Ubuntu, use o apt-get:

**$ sudo apt-get install git-all**
